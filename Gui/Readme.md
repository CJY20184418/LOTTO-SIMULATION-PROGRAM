
# Lotto

### ● Introduction
---
 이 프로그램은 C++로 구성했던 로또 시뮬레이션의 코드를 MFC를 통해 구현한 프로그램입니다.  
 Windows 환경에 적합하며, 용량은 12.8 MB입니다.  
 Lotto.exe를 설치 후 아이콘을 클릭하여 실행시키면 됩니다.
